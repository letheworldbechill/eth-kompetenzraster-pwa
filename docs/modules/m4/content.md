# Problemlösung

Problemlösung ist eine Kernkompetenz in praktisch jedem Beruf. Systematische Herangehensweise macht uns effektiver.

### Problemlösungszyklus

1. **Definition**: Problem klar beschreiben
2. **Analyse**: Ursachen identifizieren
3. **Ideation**: Lösungsansätze generieren
4. **Bewertung**: Lösungen prüfen
5. **Implementierung**: Best-Practice umsetzen
6. **Evaluation**: Ergebnisse prüfen

### Techniken

- **Brainstorming**: Ideen generieren
- **Root Cause Analysis**: Ursachen erforschen
- **Prototyping**: Schnell testen
- **Iteration**: Schrittweise verbessern

### Wichtig

- Seien Sie ausdauernd bei schwierigen Problemen
- Ein "Can-Do" Ansatz hilft
- Dokumentieren Sie Ihre Lernprozesse
